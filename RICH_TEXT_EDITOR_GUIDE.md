# 富文本编辑器使用指南

## 📝 编辑器介绍

博客系统现已集成 **Quill 富文本编辑器**，类似于哔哩哔哩专栏的编辑体验，提供所见即所得的编辑功能。

## ✨ 主要功能

### 1. 文本格式化
- **标题**：支持 H1-H6 六级标题
- **字体大小**：小号、正常、大号、超大号
- **文本样式**：
  - 粗体 (Bold)
  - 斜体 (Italic)
  - 下划线 (Underline)
  - 删除线 (Strike)

### 2. 颜色设置
- **文字颜色**：可自定义文字颜色
- **背景颜色**：可设置文字背景色

### 3. 列表和缩进
- **有序列表**：数字编号列表
- **无序列表**：项目符号列表
- **缩进控制**：增加/减少缩进

### 4. 对齐方式
- 左对齐
- 居中对齐
- 右对齐
- 两端对齐

### 5. 特殊格式
- **引用块**：用于引用文字
- **代码块**：用于显示代码
- **上标/下标**：数学公式或化学式

### 6. 媒体插入
- **链接**：插入超链接
- **图片**：插入图片（支持 URL）
- **视频**：嵌入视频（支持 YouTube、Vimeo 等）

### 7. 清除格式
- 一键清除所有格式，恢复纯文本

## 🎯 使用技巧

### 插入图片
1. 点击工具栏的图片按钮 🖼️
2. 输入图片 URL
3. 图片会自动插入到编辑器中

**推荐图床服务**：
- [Imgur](https://imgur.com) - 免费、稳定
- [SM.MS](https://sm.ms) - 国内访问快
- [PostImages](https://postimages.org) - 无需注册

### 插入代码
1. 选择"代码块"格式
2. 粘贴或输入代码
3. 代码会以深色主题显示

### 快捷键
- **Ctrl/Cmd + B**：粗体
- **Ctrl/Cmd + I**：斜体
- **Ctrl/Cmd + U**：下划线
- **Ctrl/Cmd + Z**：撤销
- **Ctrl/Cmd + Y**：重做

## 📋 预览功能

点击"👁️ 预览"按钮可以查看文章的最终显示效果，确保格式正确后再保存。

## 💾 保存说明

- 文章内容以 **HTML 格式**保存
- 保存后会在前台页面正确显示所有格式
- 支持编辑已保存的文章，格式会完整保留

## 🔄 从旧版本迁移

如果你之前使用 Markdown 编辑器创建了文章：
1. 旧文章仍然可以正常显示
2. 编辑旧文章时，内容会在富文本编辑器中打开
3. 保存后会转换为 HTML 格式

## ⚠️ 注意事项

1. **图片大小**：建议使用图床服务，避免使用 Base64 编码的大图片
2. **浏览器兼容**：建议使用现代浏览器（Chrome、Firefox、Edge、Safari）
3. **定期保存**：编辑长文章时建议定期点击保存，避免意外丢失内容
4. **复制粘贴**：从其他地方复制内容时，可能会带入额外格式，可使用"清除格式"功能

## 🎨 样式说明

编辑器中的样式会在前台页面完整呈现：
- 标题会有合适的字号和间距
- 代码块会有深色背景和语法高亮
- 引用块会有左侧边框
- 图片会自动适应宽度并居中显示
- 链接会显示为蓝色并可点击

## 🆚 与 Markdown 的区别

| 特性 | Markdown | 富文本编辑器 |
|------|----------|------------|
| 学习曲线 | 需要学习语法 | 所见即所得，无需学习 |
| 格式控制 | 有限的格式选项 | 丰富的格式选项 |
| 颜色支持 | 不支持 | 支持文字和背景色 |
| 对齐方式 | 不支持 | 支持多种对齐 |
| 使用体验 | 类似代码编辑 | 类似 Word 文档 |

## 💡 最佳实践

1. **标题层级**：合理使用标题层级，H1 用于文章标题，H2-H3 用于章节
2. **段落间距**：适当使用空行分隔段落，提高可读性
3. **图片优化**：上传前压缩图片，建议单张不超过 500KB
4. **代码展示**：使用代码块展示代码，不要用普通文本
5. **链接文字**：使用有意义的链接文字，避免"点击这里"

## 🔧 故障排除

### 编辑器无法加载
- 检查网络连接
- 确保 CDN 资源可访问
- 刷新页面重试

### 格式显示异常
- 使用"清除格式"功能重新设置
- 检查是否有嵌套的格式冲突

### 图片无法显示
- 确认图片 URL 可访问
- 检查图片链接是否使用 HTTPS
- 尝试使用其他图床服务
