# GitHub 存储快速开始

## 🎯 5 分钟配置指南

### 1️⃣ 创建 GitHub 仓库（1 分钟）

访问：https://github.com/new

```
仓库名：blog-data
可见性：Private（私有）
不要勾选任何初始化选项
```

点击"Create repository"

### 2️⃣ 生成 Token（2 分钟）

访问：https://github.com/settings/tokens/new

```
Note: Blog Data Access
Expiration: 90 days（或更长）
Scopes: ✅ repo（勾选这个）
```

点击"Generate token"，**立即复制 Token**（只显示一次！）

### 3️⃣ 配置博客（1 分钟）

1. 登录博客后台
2. 点击"GitHub 配置"
3. 填写信息：
   ```
   GitHub 用户名: 你的用户名
   仓库名: blog-data
   Token: 刚复制的 Token
   分支名: main
   ```
4. 点击"保存配置"
5. 点击"测试连接"

### 4️⃣ 推送数据（1 分钟）

点击"⬆️ 推送到 GitHub"

完成！✅

## 📱 日常使用

### 发布文章后
```
写文章 → 保存 → 推送到 GitHub
```

### 换设备时
```
配置 GitHub → 从 GitHub 拉取 → 开始使用
```

## 🔗 快捷链接

## ⚠️ 重要提示

1. **Token 只显示一次**，立即复制保存
2. **使用私有仓库**，保护数据安全
3. **定期推送**，保持数据同步
4. **Token 过期前**，提前更换
