**粘贴到编辑器：**
- 按 `Ctrl+V`
- 看到图片自动插入！✅

### 步骤 4：测试纯 URL（1 分钟）

## 🔍 故障排查

### 问题 1：粘贴后没有自动转换

**检查：**
- 是否在编辑器中粘贴？
- URL 是否包含图片扩展名？
- 浏览器控制台是否有错误？

**解决：**
- 确保在编辑器内粘贴
- 使用 Markdown 或 HTML 格式
- 查看控制台错误信息

### 问题 2：图片显示不出来

**原因：**
- 图片链接失效
- 图床限制外链
- 网络问题

**解决：**
- 检查图片链接是否有效
- 使用支持外链的图床
- 检查网络连接
